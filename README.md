# Mitosis

## How to install?
### Prerequisites
- g++
- sfml
- git
- make
### Compile & install
```shell
git clone https://github.com/jakub-swiniarski/mitosis.git
cd mitosis
sudo make install 
```
### How to run?
Run this command: 
```shell
mitosis
```
### How to uninstall?
Go inside the cloned repository and run: <br/>
```shell
sudo make clean uninstall
```

## Customization
Edit src/config.hpp and run:
```shell
sudo make install 
```
